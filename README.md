Retail Sales Analysis Using SQL
Project Overview

This project focuses on analyzing retail sales data using SQL to uncover valuable business insights related to sales performance, customer purchasing behavior, and product trends. The analysis helps businesses make data-driven decisions to improve revenue and operational efficiency.

Objectives
Analyze overall sales performance.
Identify top-selling products and categories.
Track monthly and yearly sales trends.
Understand customer purchasing patterns.
Generate actionable business insights using SQL queries.
Dataset Information

Dataset Name: Retail Sales Dataset

Key Attributes
Transaction ID
Customer ID
Product Category
Product Name
Quantity Sold
Unit Price
Total Sales Amount

Tools & Technologies
SQL
MySQL Workbench
Excel (Data Cleaning)
GitHub

Key Findings
Identified top-performing products contributing significantly to overall revenue.
Analyzed category-wise sales performance to determine profitable product segments.
Observed seasonal sales patterns through monthly trend analysis.
Identified high-value customers based on purchase behavior.
Generated insights to support strategic business decisions.
